Bootcamp Java Developer + Spring Boot

Zakres bootcampu:

Praca z systemem Linux, praca zespołowa w Scrum, praca z systemem kontroli wersji Git, programowanie w Java, programowanie w Java z użyciem frameworka Spring, budowa prostych stron internetowych, obsługa bazy danych.

Debugowanie, pisanie testów jednostkowych i integracyjnych, uruchamianie aplikacji w środowisku Docker, stosowanie podstawowych przekształceń kodu (refactoring), stosowanie podstawowych wzorców projektowych, udostępnianie i konsumowanie REST API, HTML, CSS, Java Script.
